## 📊 Structure des Titres Optimisée

### **🎯 Règle d'Or SEO**
- **H1** : 1 seule fois par page avec le mot-clé principal
- **H2** : Sous-titres avec synonymes du mot-clé principal
- **H3** : Regroupés sous chaque H2 dans une section thématique

## 🖼️ **Optimisation Images pour Performance**

### **📏 Spécifications Techniques**
- **Taille maximale** : 175-200KB par image
- **Format recommandé** : WebP > JPEG > PNG
- **Dimensions** : Adaptive avec srcset
- **Compression** : Qualité 80-85%

## ⚡ **Optimisation Vitesse de Chargement**

### **🚀 Core Web Vitals**
- **LCP (Large Contentful Paint)** : < 2.5s
- **FID (First Input Delay)** : < 100ms  
- **CLS (Cumulative Layout Shift)** : < 0.1

### **🔧 Techniques Optimisation**

#### **1. Images Optimisées**
```astro
<Image 
  src="/service-web-dev.webp" 
  alt="Service développement web Marrakech"
  widths={[640, 960, 1280]}
  sizes="(max-width: 640px) 640px, (max-width: 960px) 960px, 1280px"
  format="webp"
  loading="lazy"
/>
```

#### **2. CSS et JS Minifiés**
```javascript
// astro.config.mjs
build: {
  minify: 'terser',
  inlineStylesheets: 'auto',
}
```

#### **3. Font Loading Optimisé**
```html
<link rel="preload" as="font" type="font/woff2" href="/fonts/outfit.woff2" crossorigin>
<link rel="preload" as="font" type="font/woff2" href="/fonts/manrope.woff2" crossorigin>
```

#### **4. Critical CSS Inline**
```astro
<style is:global>
  /* CSS critique au-dessus de la ligne */
  .hero { display: flex; min-height: 100vh; }
  .cta-button { background: var(--brand-500); }
</style>
```
