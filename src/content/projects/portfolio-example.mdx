## À propos du projet

Site e-commerce moderne développé avec Astro et React, offrant une expérience d'achat fluide et intuitive. Le projet intègre **Stripe** pour les paiements sécurisés et une **gestion de panier** en temps réel.

L'architecture est optimisée pour les performances avec un **score PageSpeed Insights de 95+** et un **SEO naturel** intégré dès le départ. Le site est entièrement responsive et accessible.

## Fonctionnalités principales

- **Catalogue produit interactif** avec filtres et recherche en temps réel
- **Panier d'achat persistant** avec synchronisation locale et serveur
- **Intégration Stripe** pour les paiements sécurisés par carte et mobile
- **Dashboard administrateur** pour la gestion des produits et commandes
- **Système de notifications** en temps réel pour les utilisateurs
- **Optimisation SEO** avec balises structurées et sitemaps automatiques

## Stack technique

- **Frontend**: Astro 6 + React 19 + TypeScript
- **Styling**: Tailwind CSS v4 avec design system personnalisé
- **Backend**: Node.js + Express
- **Base de données**: PostgreSQL avec Prisma ORM
- **Paiements**: Stripe API
- **Déploiement**: Vercel avec CI/CD intégré

## Performance et accessibilité

- **Score Lighthouse**: 95+ sur tous les appareils
- **Core Web Vitals**: Vert sur toutes les métriques
- **WCAG 2.1 AA**: Conformité aux normes d'accessibilité
- **SEO optimisé**: Indexation parfaite sur Google et Bing
- **Mobile-first**: Expérience optimale sur tous les écrans
